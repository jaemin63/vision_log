# Mode Description

## Manual Mode

**기능:**
- 모든 로깅된 이미지를 리스트로 표시
- 이미지를 선택하여 우측 패널에서 Preview 가능
- 선택한 이미지를 전체 화면(Fullscreen)으로 표시 가능
- 전체 화면에서 좌우 화살표 버튼 또는 키보드(←/→)로 이전/다음 이미지 탐색 가능
- ESC 키로 전체 화면 닫기 가능

**사용 시나리오:**
- 저장된 이미지들을 수동으로 확인하고 검토할 때
- 특정 이미지를 선택하여 상세히 분석할 때
- 이미지 히스토리를 탐색할 때

---

## Auto Mode

**기능:**
- Robot 및 계측기 신호를 실시간으로 폴링
- 새로운 이미지가 생성되었을 때 자동으로 감지
- 이벤트 발생 시 화면 전면에 전체 화면으로 자동 표시
- 연속된 이벤트 발생 시 이미지가 자동으로 전환됨 (오버레이 유지)
- 전체 화면은 자동으로 유지되며, ESC 키로 닫기 불가 (이벤트가 계속 발생하므로)

**사용 시나리오:**
- Robot 작업 중 실시간으로 생성되는 이미지를 모니터링할 때
- 계측기 신호에 따라 자동으로 이미지 이벤트를 확인할 때
- 운영자가 별도 조작 없이 자동으로 이미지를 확인해야 할 때

**동작 방식:**
1. Auto Mode로 전환 시 백엔드에 폴링 시작 요청
2. 백엔드가 Modbus/TCP 또는 설정된 방식으로 Robot/계측기 신호 폴링
3. 조건 만족 시 새로운 이미지 감지
4. Socket.IO를 통해 프론트엔드에 실시간 이벤트 전송
5. 프론트엔드가 자동으로 전체 화면 오버레이 열고 이미지 표시
6. 새로운 이벤트 발생 시 기존 이미지를 새 이미지로 자동 전환

---

## Mode 전환

- **Manual → Auto**: 폴링 서비스 자동 시작, 전체 화면 오버레이 자동 표시 시작
- **Auto → Manual**: 폴링 서비스 중지, 전체 화면 오버레이 닫기

---

## 설정

Auto Mode의 폴링 설정은 Config 버튼(현재 숨김)을 통해 수정 가능:
- Modbus/TCP 설정 (IP, Port, Register 등)
- Robot PC 접속 방식 (SMB/FTP/HTTP)
- 폴링 간격 및 조건 값

설정 파일: `backend/polling-config.json`
